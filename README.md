## Configuration
Description for`.env` 

| Name        | Value                     |
|-------------|---------------------------|
| DB_NAME     | mongo database name       |
| DB_HOST     | database host             |
| DB_PORT     | database port             |
| DB_USER     | database user             |
| DB_PASSWORD | database user's password  |
| SERVER_PORT | port to launch web server |
